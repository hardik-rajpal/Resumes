\textbf{KnowYourTube}\sep\ts\sep\ang\sep\py\sep\django\sep\bs\sep{\it Personal Project } \hfill{\sl \small September 2021}\\
\vspace{-8pt}
\emerrow WebApp to analyze one's Youtube Watch History
\begin{itemize}[itemsep = -1.3 mm, leftmargin=*]
\item Formulated several functions in {\bf Python using Numpy} for filtering and organizing Youtube Watch History data collected using Google Takeout
\item Set up a server (in {\bf Python using Django}) for collecting files from clients and returning insights like view frequency over time and most watched videos among others
\item Created a user-friendly site in {\bf Angular} to allow clients to study the results with custom tags and filters
\item Deployed both the Angular and Django applications on {\bf Heroku.com}
\end{itemize}
